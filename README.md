# TheDesignUncle Portfolio Showroom

GitHub Pages 업로드용 정리 폴더입니다.

업로드할 파일과 폴더:

- `index.html`
- `gallery-data.js`
- `images/`
- `html-gallery/`
- `.nojekyll`

GitHub Pages 설정:

1. 새 저장소를 만들거나 기존 저장소를 엽니다.
2. 이 폴더 안의 내용을 저장소 최상단에 업로드합니다.
3. Settings > Pages에서 Branch를 `main`, Folder를 `/root`로 선택합니다.
4. 저장 후 제공되는 Pages 주소로 접속합니다.

`admin.html`은 공개 사이트 업로드 폴더에서 제외했습니다.
정적 호스팅에서는 서버 저장 기능이 없고, 공개 페이지에 관리 화면이 노출될 수 있기 때문입니다.
